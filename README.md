# Flat ucupaint icons

### Setup
1. Get Python
2. Get Inkscape
3. Run `pip install -r requirements.txt`

### Build process
1. Export the icon sheet at 4096x4096px as `sheet.png` into the `svg` folder
2. Run `python build.py`
